Mother Machine 这款由德国独立工作室 Maschinen-Mensch 打造的游戏被宣传为多人合作游戏领域的一股清流。仅从预告片中，你就能深刻体会到它易于上手、以社区为中心的玩法以及魔鬼般的幽默感，这些都赋予了它不可否认的魅力。虽然我不能说 Mother Machine 是一款诱饵式的“换汤不换药”游戏——它基本上做到了宣传的水平——但我还是对它失望至极。

<span>Mother Machine 是一款融合了多种元素的游戏。其简单的平台跳跃、体力管理和基于食物的增益系统，明显带有《魔兽争霸》和马力欧的基因；它略带 Roguelike 风格，利用了程序生成和随机升级；甚至还带有一丝 Edmund McMillan 式的黑色喜剧元素，尤其是在游戏开场，当游戏中的 Mother Machine 首次透露她可以无限克隆“孩子”时。</span>
<h3><span>变异系统增加了可重玩性和玩家定制</span></h3>
<img class="aligncenter size-full wp-image-127894" src="https://sky.sfcrom.com/wp-content/uploads/2025/04/202504221349516.webp" alt="" width="1200" height="675" />

<span>简单的设定通常效果最佳，尤其对于多人游戏而言。《Mother Machine》正是围绕着这个古老的设定展开，它首先会向玩家简要介绍游戏世界——一个被叛逆机器人占领的外星球——然后让他们执行一系列任务。玩家可以从自己的基地出发，独自或最多四人一起执行六种不同类型的任务。《Mother Machine》的大部分游戏玩法都发生在这些任务中，你需要完成挑战、收集升级资源，并最终激活卫星，从而推进主线剧情。</span>

<span>这些任务相当有趣，至少一开始是这样。游戏拥有出色的移动机制，如同任何优秀的平台游戏一样，每次跳跃或翻滚都蕴含着强烈的重力和动量感。游戏中也包含战斗，但远没有那么令人满足，主要就是不停地按下攻击键，偶尔翻​​滚躲避伤害。Mother Machine 并非采用线性进度模式，而是允许玩家解锁各种不同的变异技能，例如利用胀气进行二段跳或群体治疗，但每次只能激活一种。这个系统与《超级马力欧：奇迹》中的徽章系统非常相似，旨在为现有的游戏风格带来新的变化，以提升重玩性和玩家的表达能力。在掌握了这个优雅的玩法循环后，我非常期待 Mother Machine 会为我带来什么，但最终它却远远没有达到我的预期。</span>
<h3><span>有限的进度系统降低了重玩价值</span></h3>
<img class="aligncenter size-full wp-image-127895" src="https://sky.sfcrom.com/wp-content/uploads/2025/04/202504221349528.webp" alt="" width="1200" height="675" />

<span>在我11个小时的游戏时间里，我总共遇到了两种类型的Boss，包括最终Boss。要知道，这些Boss的设计没有任何变化或变化——它们实际上是两场战斗，按计划准时到来，没有任何变量可以增强它们或让它们感觉更新鲜。Misty Grove DLC是主游戏的临时免费扩展包，遗憾的是，它并没有提供更多变化：它没有引入新的任务类型，也没有提供有意义的独特关卡设计。Misty Grove本质上只是主游戏的另一种外衣。</span>

<span>Mother Machine 简陋的进阶系统背后有着明确的意图，但这种创造性的选择并没有给前面提到的乏味带来任何好处。在关卡多样性丰富的游戏中，一次只使用一种特殊能力或许效果不错，但考虑到每个关卡和游戏模式本身就只有些许差异，这种有限的机制深度只会加剧游戏的单调感。更糟糕的是，许多突变要么毫无用处，比如让玩家角色微微发光的能力，要么就是玩起来毫无乐趣，比如召唤一个环境中随处可见的投射水果的能力。在收集完所有这些突变后（这些突变甚至可能在战役进行到一半之前就发生了），收集资源的唯一奖励就只有少量的装饰品了。</span>
<h3><span>Mother Machine 的故事叙述足以弥补游戏玩法的缺陷吗？</span></h3>
<img class="aligncenter size-full wp-image-127893" src="https://sky.sfcrom.com/wp-content/uploads/2025/04/2025042213494984.webp" alt="" width="1200" height="675" />

<span>因此，在完成主线战役后，玩家就没有太多动力继续玩下去了，如果主线战役值得花钱的话，这种情况还不算太糟，但事实并非如此。主线战役由五个故事任务组成，每个任务之间间隔着越来越大的卫星需求，也就是常见的重复性玩法。这五个任务相当简洁，每个任务的持续时间与标准任务差不多，这让人很失望。但真正的问题是，它们在机制上与常规任务没有区别：玩家必须导航到各个气闸并通过一些挑战，这些挑战也与其他任务中的挑战相同。</span>

剧情任务与普通任务之间唯一的主要区别在于，它们会通过 Mother Machine 的对话和各种文本日志来提供一些信息。但这种叙事远不够充实，也缺乏创意，不足以弥补 Mother Machine 玩法留下的巨大缺陷。你可以对 Maschinen-Mensch 的其他不足之处给予宽容——毕竟它是一家小型独立工作室——但由于剧情主要通过文本展开，感觉像是在这一点上有所疏漏。创作引人入胜的故事并不需要 3A 级的预算，虽然大多数人不会为了刺激的剧情而玩这类游戏，但引人入胜的故事或许可以弥补玩法上的不足。
